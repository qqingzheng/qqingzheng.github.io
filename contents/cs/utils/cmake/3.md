# 编译动态库和静态库

### 文件结构

├── bin
│   └── example
├── CMakeLists.txt
├── include
│   ├── Add.h
│   └── Minus.h
├── lib
└── src
    ├── Add.cpp
    ├── Example.cpp
    └── Minus.cpp

我们将Add输出为库文件

### CMakeLists.txt

```
cmake_minimum_required(VERSION 3.10)
project(demo)
include_directories(include)
add_library(add_shared SHARED src/Add.cpp)
add_library(add_static STATIC src/Add.cpp)
set_target_properties (add_shared PROPERTIES OUTPUT_NAME "add")
set_target_properties (add_static PROPERTIES OUTPUT_NAME "add")
set (LIBRARY_OUTPUT_PATH ${PROJECT_SOURCE_DIR}/lib)
```

+ add_library(<目标名> <SHARED/STATIC> 源文件)
+ set_target_properties(<目标名> PROPERTIES <属性> <值>)

属性：OUTPUT_NAME-输出的名字；VERSION-版本

+ LIBRARY_OUTPUT_PATH 输出库的目录
