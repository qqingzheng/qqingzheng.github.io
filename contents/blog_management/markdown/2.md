# 1，段落

### 换行

我们只需要让两个段落之间有一个以上的空行就可以了，比如：

	我要学MarkDown
	空行
	MarkDown好难

那么在网页上就会显示：

我要学MarkDown

MarkDown好难

# 2，标题

### 基本概念
标题分为六阶，**一阶最大，六阶最小**。写标题有两种方法，一种是下划线法，一种是#号法。
### 下划线法
**用法**：在标题下方一行输入"======="或"---------"即可。例子如下：

	我是一阶标题
	============
	我是二阶标题
	-----------


**局限**: 下划线法只能用于一阶标题和二阶标题。
### #号法
**用法**: 在标题前方加#号，**几阶标题加几个**，还要用空格与标题隔开。例子如下：

	# 我是一阶标题
	## 我是二阶标题
	### 我是三阶标题
	...
