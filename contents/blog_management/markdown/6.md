代码、表格
===================
-------------
# 1，代码

### 方法1
**用法**: 在一行前面加四个空格或Tab

		printf("hello world!");

**效果**: 

	printf("hello world!");
	
### 方法2

**用法**: 用`(反引号)括起来。例如：

	`printf("hello world!");`
	
**效果**: `printf("hello world!");`

**拓展用法**: 指定编程语言。例如：

	```c++
	printf("hello world!");`
	```
**效果**: 

```C++
printf("hello world!");
```

# 2，表格

**用法**: 

	|  标题1   | 标题2  |
	|  ----  | ----  |
	| 内容1  | 内容3 |
	| 内容2  | 内容4 |

**效果**: 

|  标题1   | 标题2  |
|  ----  | ----  |
| 内容1  | 内容3 |
| 内容2  | 内容4 |

-------------
# 3，表格内容的位置
**用法**: 

	居中:
	
	|  标题1   | 标题2  |
	|  :----:  | :----:  |
	| 内容1  | 内容3 |
	| 内容2  | 内容4 |
	
	居左
	
	|  标题1   | 标题2  |
	|  :----  | :----  |
	| 内容1  | 内容3 |
	| 内容2  | 内容4 |
	
	居右
	
	|  标题1   | 标题2  |
	|  ----:  | ----:  |
	| 内容1  | 内容3 |
	| 内容2  | 内容4 |
	
**效果**: 仅展示居右。

|  我是很长很长很长的标题1   | 我是很长很长很长的标题2  |
|  ----:  | ----:  |
| 内容1  | 内容3 |
| 内容2  | 内容4 |
